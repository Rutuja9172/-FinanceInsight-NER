# FinanceInsight-NER
## Project Overview
FinanceInsight-NER is a project focused on building a Named Entity Recognition (NER) system for financial text. The goal is to identify useful financial information from unstructured text such as company names, monetary values, dates, and percentages.

---

## What is Named Entity Recognition (NER)?
Named Entity Recognition (NER) is a Natural Language Processing (NLP) technique used to identify and classify important elements in text into predefined categories.

---

## Financial Entities Covered
The project aims to identify the following entity types:
- Organization (ORG)
- Monetary Value (MONEY)
- Date (DATE)
- Percentage (PERCENT)

---

## Example
**Input Text:**  
Tata Motors reported revenue of ₹3,200 crore in Q3 2024.

**Identified Entities:**  
- Tata Motors → ORG  
- ₹3,200 crore → MONEY  
- Q3 2024 → DATE
